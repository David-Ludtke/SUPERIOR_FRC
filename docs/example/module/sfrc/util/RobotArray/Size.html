sfrc::Util::RobotArray::Size:

size_t Size()

return's the current size of the
RobotArray. cannot fail

Examples:
  • 
    sfrc::Util::RobotArray<int> arr = sfrc::Util::RobotArray<int>::Create();
    arr.PushBackArray(new int[3]{1, 2, 3}, 3);
    size_t size = arr.Size(); // size is 3
                    
  • 
    sfrc::Util::RobotArray<std::string> strArr = sfrc::Util::RobotArray<std::string>::OfVec({"a", "b", "c", "d"});
    size_t strSize = strArr.Size(); // strSize is 4
