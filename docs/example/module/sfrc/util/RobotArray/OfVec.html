sfrc::Util::RobotArray::OfVec:

static RobotArray<T> OfVec(std::vector<T> vec)

this function takes a vector
and turns it into a robot array
this function can fail but only
in extreme memory shortage cases
the size of the New RobotArray
will be equal to the size of the vector
however the capacity size is deterministic
but should not be relied on because
it may change between versions
has an overload that
takes in a initializer list for easier use

Examples:
  • 
    sfrc::Util::RobotArray<int> myArray = sfrc::Util::RobotArray<int>::OfVec({1, 2, 3, 4, 5});
    //creates a RobotArray of integers with the values 1, 2, 3, 4, 5
    myArray.Size(); //returns 5
                    
  • 
    std::vector<std::string> strVec = {"hello", "world", "from", "RobotArray"};
    sfrc::Util::RobotArray<std::string> strArray = sfrc::Util::RobotArray<std::string>::OfVec(strVec);
    //creates a RobotArray of strings with the values from strVec
