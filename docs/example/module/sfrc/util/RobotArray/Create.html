sfrc::Util::RobotArray::Create:

static RobotArray Create()

Creates an empty RobotArray and
Returns it to the caller
this function cannot fail

Examples:
  • 
    sfrc::Util::RobotArray<int> myArray = sfrc::Util::RobotArray<int>::Create();
    //creates an empty RobotArray of integers
    myArray.Size(); //returns 0
                    
  • 
    sfrc::Util::RobotArray<std::string> strArray = sfrc::Util::RobotArray<std::string>::Create();
    //creates an empty RobotArray of strings
    strArray.Size(); //returns 0
