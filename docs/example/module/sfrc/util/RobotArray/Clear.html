sfrc::util::RobotArray::Clear:

void Clear()

it clears the RobotArray, removing all elements and resetting its size to zero.
it is effectivly the same as creating a new RobotArray with RobotArray::create.
anything like the robots capacity is left in a detirministic state, but undefined
state to the caller after the clear operation. this function will never fail.

Examples:
  • 
    sfrc::Util::RobotArray<int> arr = sfrc::Util::RobotArray<int>::Create();
    arr.PushBackArray(new int[3]{1, 2, 3}, 3);
    arr.Clear();
    // arr is now empty and its size is 0
    arr.At(0) // returns none
                    
  • 
    sfrc::Util::RobotArray<std::string> strArr = sfrc::Util::RobotArray<std::string>::OfVec({"hello", "world"});
    strArr.Clear();
    // strArr is now empty
    strArr.Size() // returns 0
