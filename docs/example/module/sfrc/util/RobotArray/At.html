sfrc::util::RobotArray::At:

std::Optional<T> At(size_t index)

At checks the index to see if it's within
the valid range of indices if it's not
then it return's none else it returns
the element at the chosen index

Examples:
  • 
sfrc::util::RobotArray<int> array = sfrc::util::RobotArray<int>::Create();
array.PushBackArray(new int[3]{10, 20, 30}, 3);
std::Optional<int> value = array.At(1); // value contains 20
std::Optional<int> invalidValue = array.At(5); // invalidValue contains none
                    
  • 
sfrc::util::RobotArray<std::string> stringArray = sfrc::util::RobotArray<std::string>::OfVec({"apple", "banana", "cherry"});
std::Optional<std::string> fruit = stringArray.At(0); // fruit contains "apple"
std::Optional<std::string> noFruit = stringArray.At(10); // noFruit contains none
