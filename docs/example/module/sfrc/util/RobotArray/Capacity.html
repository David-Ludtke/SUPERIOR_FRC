sfrc::util::RobotArray::Capacity:

size_t Capacity()

it returns the current capacity of the RobotArray
this function cannot fail

Examples:
  • 
    sfrc::Util::RobotArray<int> arr = sfrc::Util::RobotArray<int>::Create();
    arr.PushBackArray(new int[5]{1, 2, 3, 4, 5}, 5);
    size_t capacity = 5; // capacity is 5
                    
  • 
    sfrc::Util::RobotArray<std::string> strArr = sfrc::Util::RobotArray<std::string>::OfVec({"a", "b", "c"});
    size_t strCapacity = strArr.Capacity(); // strCapacity is 3
