sfrc::util::SwapPtr:

void SwapPtr(T* ptr1, T* ptr)

SwapPtr takes 2 pointers and swaps
the underlying data they point to.
it does not swap the pointers themselves.
but it swaps the data they point to.
it does not do a nullptr check so be sure to
only pass valid pointers to it.
this function is templated so it can work with any type.
however both pointers must be of the same type.

Examples:
  • 
    SwapPtr(ptrA, ptrB);
    //swaps the data pointed to by ptrA and ptrB
    //now ptrA points to the data that was originally pointed to by ptrB and vice versa.
                    
  • 
    SwapPtr(strPtr1, strPtr2);
    //swaps the strings pointed to by strPtr1 and strPtr2
    //now strPtr1 points to the string that was originally pointed to by strPtr2 and vice versa.
